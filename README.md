# Runner

A 2D platformer game where the player avoids obstacles, collects coins, and tries to reach the end of the level.

## Preview
A 2D platformer game where the player avoids obstacles, collects coins, and tries to reach the end of the level.

![Game GIF](./assets/screenshot.png)

## About the Game
- Genre: Platformer / Minimalist.
- Objective: The Player need to avoid the traps and reach the gate to complete the level.
- Gameplay: Movement, jumping.

## Features
- Player movement
- Enemies / obstacles
- Multiple levels
- Sound effects
- Game over / restart system

## Built With
- Language: C#
- Engine / Framework: Unity.
- Tools: GitHub, pixilart.

## How to Run

### Requirements
Example:
- Unity 2023.2.20f1
- Visual Studio 

### How to Run ?
- Open the Project in Unity and press "Play".
